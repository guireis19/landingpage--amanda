# 🌻 Amanda de Osún — Landing Page

> Landing page institucional com foco em captação de clientes via redes sociais, direcionando para contato pelo WhatsApp.

---

## 📋 Sobre o Projeto

Site desenvolvido para **Amanda de Osún**, mãe de santo com mais de 8 anos de tradição africanista em Porto Alegre. O objetivo principal é converter visitantes vindos de redes sociais (Instagram, TikTok etc.) em clientes reais através de um CTA direto para o WhatsApp.

**URL em produção:** [amandadeosun.com.br](https://amandadeosun.com.br/)

---

## 🎯 Objetivo do Cliente

Captar clientes que chegam via redes sociais e direcioná-los para entrar em contato pelo site — centralizando o primeiro contato pelo WhatsApp, com uma apresentação profissional dos serviços oferecidos.

---

## ✨ Funcionalidades

- Navegação suave entre seções (âncoras)
- Botão fixo de WhatsApp para contato rápido
- Seção de apresentação com estatísticas (8+ anos, 15k+ vidas impactadas)
- Cards de serviços oferecidos (Jogo de Búzios, Tarô, Consultas Online)
- Seção de depoimentos de clientes
- Formulário / seção de contato
- Design responsivo (mobile-first)
- CTA principal direcionado ao WhatsApp
